# Repositório de Aulas de Python

Este repositório contém exercícios e exemplos de código do curso de Python, organizados por aula.

## Resumo das Aulas

### Aula 1: `print()` e Comentários

- **Conceitos:** Introdução à função `print()` para exibir informações na tela e a importância dos comentários (`#` para linha única e `"""..."""` para múltiplas linhas) para documentar e explicar o código.

### Aula 2: Divisão Inteira (`//`)

- **Conceitos:** Apresentação do operador de divisão inteira (`//`), que realiza uma divisão e retorna apenas a parte inteira do resultado, descartando qualquer casa decimal.

### Aula 3: Variáveis e Operadores Aritméticos

- **Conceitos:** Introdução ao uso de variáveis para armazenar dados. Demonstração dos operadores aritméticos básicos: soma (`+`), subtração (`-`), multiplicação (`*`), divisão (`/`), módulo (`%`) e potenciação (`**`).

### Aula 4: Tipos de Dados

- **Conceitos:** Explicação sobre os tipos de dados primitivos em Python: `str` (texto), `int` (inteiro), `float` (número com decimal) e `bool` (booleano). Apresenta a função `type()` para verificar o tipo de uma variável.

### Aula 5: Exercício Prático de Variáveis

- **Conceitos:** Exercício para fixar o conhecimento sobre variáveis e tipos de dados, criando um pequeno cadastro com nome, idade, altura e se é maior de idade.

### Aula 6: Concatenação e Repetição de Strings

- **Conceitos:** Demonstra como o operador `+` pode ser usado para concatenar (juntar) strings e como o operador `*` pode ser usado para repetir uma string um determinado número de vezes.

### Aula 7: Precedência de Operadores

- **Conceitos:** Explica a ordem em que o Python resolve as operações matemáticas em uma expressão (parênteses, potenciação, multiplicação/divisão e, por último, soma/subtração).

### Aula 8: Exercício - Cálculo do IMC

- **Conceitos:** Exercício prático que utiliza variáveis e a fórmula `Peso / (Altura ** 2)` para calcular o Índice de Massa Corporal (IMC) de uma pessoa.

### Aula 9: Introdução a F-Strings

- **Conceitos:** Apresenta as f-strings (`f"..."`) como uma maneira moderna e legível de formatar strings, permitindo embutir variáveis e expressões diretamente no texto.

### Aula 10: Formatação de Números com F-Strings

- **Conceitos:** Aprofunda o uso de f-strings para formatar a exibição de números, controlando o número de casas decimais (`:.2f`), adicionando zeros à esquerda e formatando como moeda.

### Aula 11: Validação de Entrada com `.isdigit()`

- **Conceitos:** Ensina como usar o método `.isdigit()` para verificar se a entrada do usuário contém apenas dígitos numéricos antes de tentar convertê-la para um número, evitando erros.

### Aula 12: Modificadores de Formatação em F-Strings

- **Conceitos:** Detalha como usar modificadores em f-strings para alinhar o texto à direita (`>`), à esquerda (`<`), ou ao centro (`^`), e preencher os espaços vazios com caracteres específicos.

### Aula 13: Manipulação de Strings (Índices e Fatiamento)

- **Conceitos:** Aborda como acessar caracteres individuais de uma string usando índices e como extrair "pedaços" (substrings) usando fatiamento (`string[inicio:fim:passo]`). Apresenta a função `len()`.

### Aula 14: Laço de Repetição `while`

- **Conceitos:** Introduz o laço `while`, que executa um bloco de código repetidamente enquanto uma condição for verdadeira. Demonstra o uso de contadores e da palavra-chave `break` para sair do laço.

### Aula 15: Laço de Repetição `for` e `range()`

- **Conceitos:** Apresenta o laço `for`, ideal para iterar sobre sequências (como strings ou listas). Mostra como usar a função `range()` para gerar sequências numéricas para o `for`.

### Aula 16: `continue` e `for/else`

- **Conceitos:** Explica a palavra-chave `continue` para pular para a próxima iteração de um laço e a estrutura `for/else`, onde o bloco `else` é executado se o laço terminar sem ser interrompido por um `break`.

### Aula 17: Métodos `split()`, `join()` e `enumerate()`

- **Conceitos:** Apresenta três recursos poderosos: `.split()` para dividir uma string em uma lista, `.join()` para unir uma lista em uma string, e `enumerate()` para obter o índice e o valor ao iterar.

### Aula 18: Desempacotamento de Listas

- **Conceitos:** Mostra como atribuir os itens de uma lista a múltiplas variáveis de uma só vez. Introduz o uso do `*` para capturar múltiplos itens restantes em uma nova lista.

### Aula 19: Operador Ternário

- **Conceitos:** Apresenta o operador condicional ternário (`valor_se_verdadeiro if condicao else valor_se_falso`), uma forma compacta de escrever uma instrução `if-else` em uma única linha.

### Aula 20: Curto-Circuito com `or` e `and`

- **Conceitos:** Explica como os operadores `or` e `and` funcionam com avaliação de curto-circuito, retornando um dos valores da expressão em vez de apenas `True` ou `False`.

### Aula 21: Revisão de Laços

- **Conceitos:** Aula de revisão que combina `for`, `range`, e `enumerate` em um único script para criar uma contagem regressiva que também exibe o índice de cada passo da iteração.

### Aula 22: Validador de CPF

- **Conceitos:** Projeto prático que implementa um validador de CPF completo, aplicando as regras matemáticas oficiais para o cálculo dos dígitos verificadores e checando por sequências inválidas.

### Aula 23: Funções (`def`)

- **Conceitos:** Introdução à criação de funções com a palavra-chave `def`. Explica como definir parâmetros com valores padrão e como usar a instrução `return` para enviar um resultado de volta.
