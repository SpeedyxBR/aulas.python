# Aulas de Python

Este repositório contém o código-fonte e os exercícios do curso de Python, organizados por aula.

## Conteúdo das Aulas

- **Aula 1:** Introdução ao `print()` e comentários
- **Aula 2:** Operador de divisão inteira (`//`)
- **Aula 3:** Variáveis e operadores aritméticos
- **Aula 4:** Tipos de dados e a função `type()`
- **Aula 5:** Exercício de variáveis e tipos de dados
- **Aula 6:** Concatenação e repetição de strings
- **Aula 7:** Precedência de operadores aritméticos
- **Aula 8:** Exercício de cálculo do IMC
- **Aula 9:** Introdução à formatação de strings com f-strings
- **Aula 10:** Formatação de números com f-strings
- **Aula 11:** Usando o método `format()` para strings
- **Aula 12:** Entrada de dados com `input()`
- **Aula 13:** Exercício de calculadora simples com `input()`
- **Aula 14:** Introdução às estruturas condicionais `if`, `elif`, `else`
- **Aula 15:** Operadores relacionais (lógicos)
- **Aula 16:** Operadores lógicos `and`, `or`, `not`
- **Aula 17:** Exercício de sistema de login simples
- **Aula 18:** Interpolação de strings com o operador `%`
- **Aula 19:** Exercício de validação de dados de entrada
- **Aula 20:** Tratamento de erros com `try` e `except`
- **Aula 21:** Desempacotamento de listas
